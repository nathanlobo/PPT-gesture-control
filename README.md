# 🚀 AirType – Gesture Controlled Keyboard

AirType is a real-time AI-based virtual keyboard that allows users to type using hand gestures without touching any physical device.

## 🧠 Features
- ✋ Hand tracking using MediaPipe
- ⌨️ Virtual keyboard UI
- 🤏 Pinch gesture for typing
- 🔙 Backspace support
- 🌐 Works with real applications (Chrome, Notepad, etc.)

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## 🎯 How It Works
Camera → Hand Detection → Finger Tracking → Gesture Recognition → Typing Output
